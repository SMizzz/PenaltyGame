# Penalty Game

iOS 스터디에서 첫 번째로 제작한 토이 프로젝트입니다.
페널티 게임이라는 작품이며, 인원수를 입력받아 랜덤으로 페널티를 받을 사람을 정하는 게임입니다.
각자 코드 스타일대로 구현했습니다.
